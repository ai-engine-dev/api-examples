# NSFW Detect

Detect NSFW/inappropriate content in images for content moderation.

- **Endpoint**: `POST https://nsfw-detect3.p.rapidapi.com/nsfw-detect`
- **Response: JSON with body.ModerationLabels[] — empty array means safe content**
- [RapidAPI Page](https://rapidapi.com/ai-engine-ai-engine-default/api/nsfw-detect3)
- [Full Documentation](https://ai-engine.net/apis/nsfw-detect)

## Getting an API Key

1. Go to [NSFW Detect on RapidAPI](https://rapidapi.com/ai-engine-ai-engine-default/api/nsfw-detect3)
2. Click **Subscribe** (free tier available)
3. Copy your API key and replace `YOUR_API_KEY` in the examples
