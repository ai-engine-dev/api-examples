# Emotion & Personality Analysis

Detect emotions and personality traits from text.

- **Endpoint**: `POST https://emotion-sentiment-personality-analysis.p.rapidapi.com/emotions`
- **Response**: JSON with `emotions[]` containing `label` and `score` (0-1)
- [RapidAPI Page](https://rapidapi.com/ai-engine-ai-engine-default/api/emotion-sentiment-personality-analysis)
- [Full Documentation](https://ai-engine.net/apis/emotion-analysis)

## Getting an API Key

1. Go to [Emotion & Personality Analysis on RapidAPI](https://rapidapi.com/ai-engine-ai-engine-default/api/emotion-sentiment-personality-analysis)
2. Click **Subscribe** (free tier available)
3. Copy your API key and replace `YOUR_API_KEY` in the examples
