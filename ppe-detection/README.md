# PPE Detection

Detect personal protective equipment (helmets, masks, gloves) on people.

- **Endpoint**: `POST https://ppe-detection1.p.rapidapi.com/detect-ppe`
- **Response: JSON with body.Persons[] containing HeadCover, FaceCover, hand covers with confidence**
- [RapidAPI Page](https://rapidapi.com/ai-engine-ai-engine-default/api/ppe-detection1)
- [Full Documentation](https://ai-engine.net/apis/ppe-detection)

## Getting an API Key

1. Go to [PPE Detection on RapidAPI](https://rapidapi.com/ai-engine-ai-engine-default/api/ppe-detection1)
2. Click **Subscribe** (free tier available)
3. Copy your API key and replace `YOUR_API_KEY` in the examples
